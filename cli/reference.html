Option Types Formatting¶

General Options¶

--workers
The number of concurrent downloads when requesting multiple scenes. - Default 4
--verbose
Specify verbosity
--api-key
Valid API key - or via ENV variable PL_API_KEY
--base-url
Change the base Planet API URL or ENV PL_API_BASE_URL - Default https://api.planet.com/
--version
Show the version and exit.

download¶

Download quads from a mosaic

Usage: download [OPTIONS] NAME

Options¶
Name Description Format
rbox Region to download as a comma-delimited string: lon_min,lat_min,lon_max,lat_max RBOX
quiet Disable ANSI control output BOOLEAN
dest

Location to download files to

DEFAULT: .

 DIRECTORY
limit Limit the number of items. NUMBER
